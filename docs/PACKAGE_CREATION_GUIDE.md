# Package Creation Guide

Quick reference for creating packages in the Ottabase Turborepo monorepo.

## Core Principles

- **Keep packages generic** - Reusable across different apps
- **React dependencies allowed** - Packages can use React/React DOM as peer dependencies
- **No Next.js specifics** - Next.js-specific code belongs in app directories (e.g.
  `apps/ottabase-template-app-nextjs-homepage`)

## Package Structure

```text
packages/package-name/
├── package.json
├── tsconfig.json
├── src/
│   ├── index.ts              # Barrel exports
│   ├── component-name.ts     # Atomic exports
│   └── components/
└── dist/                     # Generated by build; .gitignore'd
```

## Package.json Template

```jsonc
{
    "name": "@ottabase/package-name",
    "version": "1.0.0",
    "main": "dist/index.js",
    "module": "dist/index.esm.js",
    "types": "dist/index.d.ts",
    "exports": {
        ".": {
            "types": "./dist/index.d.ts",
            "import": "./dist/index.esm.js",
            "require": "./dist/index.js",
        },
        "./component-name": {
            "types": "./dist/component-name.d.ts",
            "import": "./dist/component-name.esm.js",
            "require": "./dist/component-name.js",
        },
    },
    "sideEffects": false, // Enable tree shaking when possible
    "scripts": {
        "build": "tsup src/index.ts src/component-name.ts --format cjs,esm --dts --clean",
        "dev": "tsup src/index.ts src/component-name.ts --format cjs,esm --dts --watch",
    },
    "dependencies": {
        "@ottabase/config": "workspace:*",
    },
    "peerDependencies": {
        "react": ">=18.0.0",
        "react-dom": ">=18.0.0",
    },
    "devDependencies": {
        "typescript": "catalog:",
        "tsup": "catalog:",
    },
}
```

## TypeScript Configuration

```json
{
    "extends": "../../tsconfig.json",
    "compilerOptions": {
        "outDir": "dist",
        "jsx": "react-jsx",
        "skipLibCheck": true
    },
    "include": ["src/**/*"],
    "exclude": ["dist", "node_modules"]
}
```

## Atomic Imports (Tree Shaking)

Create individual entry files for better tree shaking:

```typescript
// src/component-name.ts
export { default as ComponentName } from './components/ComponentName';
```

Add to package.json exports and build script:

```json
{
    "exports": {
        "./component-name": {
            "types": "./dist/component-name.d.ts",
            "import": "./dist/component-name.esm.js",
            "require": "./dist/component-name.js"
        }
    },
    "scripts": {
        "build": "tsup src/index.ts src/component-name.ts --format cjs,esm --dts --clean"
    }
}
```

Usage:

```typescript
// Atomic import (better tree shaking)
import { ComponentName } from '@ottabase/package-name/component-name';

// Barrel import (convenience)
import { ComponentName } from '@ottabase/package-name';
```

## Tailwind Integration

Consumer apps must include package source in Tailwind config:

```javascript
// tailwind.config.cjs
module.exports = {
    content: ['./app/**/*.{js,ts,jsx,tsx}', '../../packages/package-name/src/**/*.{js,ts,jsx,tsx}'],
};
```

## Dependencies

- **Workspace packages**: `"@ottabase/config": "workspace:*"`
- **Catalog packages**: `"lucide-react": "catalog:"`
- **Peer dependencies**: React/React DOM (avoid Next.js peer deps)

Use `catalog:` for shared dependencies defined in pnpm-workspace.yaml (e.g., TypeScript, tsup). Use `workspace:*` for
internal packages. Declare framework dependencies (e.g., React) as peerDependencies to avoid duplication

## Development Workflow

```bash
pnpm install
pnpm build
pnpm dev  # Watch mode
```

## Testing

Test components in the primary app (`apps/ottabase-template-app-tanstack`):

```typescript
// apps/ottabase-template-app-tanstack/src/pages/test-page.tsx
import { ComponentName } from '@ottabase/package-name/component-name';

<ComponentName prop1="value" />
```

## Key Reminders

- Packages should be **framework-agnostic** as much as possible
- App-specific code goes in the relevant app directory (primary: `apps/ottabase-template-app-tanstack`)
- Remove `rootDir` from tsconfig, use `skipLibCheck: true`
- Include package sources in consumer app Tailwind config
