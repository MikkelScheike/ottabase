# @ottabase/ui-code-highlight

Code Highlight component and provider for Ottabase applications.

## Installation

```sh
pnpm add @ottabase/ui-code-highlight
```

## Usage

```tsx
import { ProviderCodeHighlight } from '@ottabase/ui-code-highlight';

function App() {
    return <ProviderCodeHighlight>{/* Your app */}</ProviderCodeHighlight>;
}
```
